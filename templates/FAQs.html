FREQUENTLY ASKED QUESTIONS

  • The project is designed to identify the language of a given sentence, perform part-of-speech (POS) tagging, and detect profanity. It first loads or trains a language identification model, which predicts the language based on the input sentence. Once the language is identified, the appropriate POS tagger processes the sentence to assign grammatical tags to each word. The project also includes a profanity detection component that uses a trained Support Vector Machine (SVM) model to classify sentences as profane or not, and utilizing n-grams derived from the POS tags to identify patterns associated with profanity. If a sentence is detected as profane, it is censored accordingly. Finally, the results, including the detected language, POS tags, and any censored sentences, are displayed.

  • Naive Bayes for the Language Identification.

    Stanford POS Tagger for the Part-of-Speech Tagging.

    Support Vector Machine and N-gram for the Profanity Detection and Pattern Generation.
